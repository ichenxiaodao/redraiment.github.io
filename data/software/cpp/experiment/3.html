三、学校有近千名学生，在操场上排队，5人一行余2人，7人一行余3人，3人一行余1人，编写一个程序求该校的学生人数。

要求：请同学把预备知识、步骤、程序框图、调试好的程序及存在的问题写在下面（不够可以附页）。

#1.c

#include <stdlib.h>
#include <stdio.h>

int main(void) {
  int x, y, z;

  for (z = 1; z <= 6; z += 5) {
    y = (13 - z) % 10 - 2 * z / 10;
    x = 4 + (13 - z) / 10 - y;
    if (0 <= x && x <= 9
     && 0 <= y && y <= 9
     && x != y && x != z
     && y != z) {
      printf("x=%d y=%d z=%d\n", x, y, z);
    }
  }

  return EXIT_SUCCESS;
}

#2.c

#include <stdlib.h>
#include <stdio.h>
#include <math.h>

int main(void) {
  int x, y;

  for (y = -3; y <= 3; y++) {
    printf("%*d", 4 - abs(y), 1 + abs(y));
    for (x = 0; x < 2 * abs(y); x++) {
      printf("%d", 1 + abs(y));
    }
    putchar('\n');
  }

  return EXIT_SUCCESS;
}

#3.c

#include <stdlib.h>
#include <stdio.h>
#include <math.h>

int main(void) {
  int n;

  for (n = 1;
       n % 5 != 2
    || n % 7 != 3
    || n % 3 != 1;
       n++);
  printf("%d\n", n);

  return EXIT_SUCCESS;
}
