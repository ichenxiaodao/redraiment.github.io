Problem Description

Give you a number on base ten,you should output it on base two.(0 < n < 1000)

Input

For each case there is a postive number n on base ten, end of file.

Output

For each case output a number on base two.

Sample Input

1
2
3

Sample Output

1
10
11

问题分析

Problem Analyse

给你一个十进制的数，输出该数的二进制表示方式。

Algorithm Analyse

数制的转换知识参看2031。

参考源码

#include <stdio.h>

void TtoB(int n)
{
  if (n)
  {
    TtoB(n >> 1);
    printf("%d", n & 1);
  }
}

int main(void)
{
  int n;
  while (scanf("%d", &n) != EOF)
  {
    TtoB(n);
    putchar('\n');
  }

  return 0;
}
