问题描述

Problem Description

给定三条边，请你判断一下能不能组成一个三角形。

Input

输入数据第一行包含一个数M，接下有M行，每行一个实例，包含三个正数A,B,C。其中A,B,C <1000;

Output

对于每个测试实例，如果三条边长A,B,C能组成三角形的话，输出YES，否则NO。

Sample Input

2
1 2 3
2 2 2

Sample Output

NO
YES

问题分析

Problem Analyse

基本数学知识

Algorithm Analyse

如果a, b, c三条边能组成三角形，则

a + b > c

a + c > b

b + c > a

算法实现

读数据的时候，不要忘了三条边要用实数型，而不是整型。^_^

参考源码

#include <stdio.h>

int main(void)
{
  int n;
  double a, b, c;

  scanf("%d", &n);
  while (n-- && scanf("%lf%lf%lf", &a, &b, &c))
    puts(a + b > c && a + c > b && b + c > a ? "YES" : "NO");

  return 0;
}
