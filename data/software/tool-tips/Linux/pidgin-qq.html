很多同学不愿意从Windows过渡到Linux，很大一个原因是习惯了Windows下的一些应用程序。比如腾讯的QQ，看来马化腾的QQ比Linux更有吸引力。

现在买了Fedora 6.0，打算开始脱离Windows了（这让我想起了美国记者说1年不用中国货）。早在我买Fedora前，我就在网上查询过Linux下QQ的信息，以前大家比较推崇的是LamaQQ，但我下载过来后却用不了（能运行，就是点登入后没反应）。

今天来了兴致，又去网上搜索了一下，发现一款叫Gaim的软件。马上去Baidu一下它的信息，据说现在已经更名为“Pidgin”了，找到了它的官方网站 http://www.pidgin.im/ 页面很简洁。马上进入Download页面，下载了最新的源代码包，解压后就可以用。

看到还有个“Fedora Core”的下载页面，莫非有rpm包？点进入看看，信息如下：

To install Pidgin on Fedora Core 4, 5 or 6, save the pidgin.repo file from above in /etc/yum.repos.d/
You can then use "yum install pidgin", or the Package Manager (pirut) to install Pidgin.
We do not provide packages for Fedora 7. If you use Fedora 7, please wait for the Fedora developers to release updated packages.

上面提示说把文件 http://rpm.pidgin.im/fedora/pidgin.repo 下载下来，放到目录 /etc/yum.repos.d/ 下，然后在解压出来的Pidgin文件目录下运行"yum install pidgin"就开始自动安装了。我实验了一下，果然可以！安装完以后在目录下双击运行就可以了。

没想到这次这么轻松就搞定了～HOHO。
